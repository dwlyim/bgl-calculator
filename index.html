Security

While it doesn't appear you can see formulas etc, a tech savvy person can actually still access the underlying Excel. Although the Excel is locked and relevant formulas and price lists hidden, tools and methods exist to break the security. So it's not bulletproof against somebody who really wants to break in.

Having said that, even if the security was bulletproof, a sufficiently determined person could still simply continuously plug in numbers to the calculator, note the results, and eventually reverse engineer the price list.

The other thing to note is that if you want to edit the underlying Excel, you will need to first unlock it with a password, and then reapply the password once you are done editing.
